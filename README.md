# HorseRaceSimulator
A horse racing simulator with PHP and Symfony Framework.

# Instructions
## Build the docker containers
```bash
cd docker/ && docker-compose up -d --build --remove-orphans
```

### Check docker logs
```bash
docker-compose logs -f
```
